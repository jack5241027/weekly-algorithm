# 206. Reverse Linked List

## [題目連結](https://leetcode.com/problems/reverse-linked-list/submissions/)

## Code

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
var reverseList = function(head) {
  let prev = null;

  while (head !== null) {
    const temp = head.next;
    head.next = prev;
    prev = head;
    head = temp;
  }

  return prev;
};
```
